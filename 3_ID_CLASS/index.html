ID e CLASS

ID

O ID funciona como um cps. Assim como so posso ter um numero de CPF, cada elemento ao pode ter um valor de ID. O CPS é o único, ou seja, ninguem pode ter um CPF igual o meu. Isso significa que o ID tambem é único. entao nao é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que so pode ter um valor, no CLASS voce pode ter varios valores. Tambem diferente do ID, em que nao se pode repetir um valor, no CLASS voce pode utilizar o mesmo valor pra vários elementos.
